This initial project serves mainly to demonstrate basic skills with Python and Jupyter notebook, importing data using Pandas and running basic classification algorithms using scikit-learn. Remember that all code has to be your own, and all documentation and analysis text needs to be your own words.

1. Preliminaries: Python and Jupyter Notebook

As discussed in class, on your own machine you need to install Python 3.10, and use pip to install the following packages:

  • scikit-learn
  • pandas
  • xgboost
  • jupyter

If you already have an Anaconda distribution installed, and you know how to use it to install packages, you are welcome to use that as well.

All of the steps below should be shown in a Jupyter notebook file that you will submit.

You should use at least one separate notebook cell for each of the four task sections below. You may divide each section up further into additional cells if you wish. Just make sure that all the required information is printed out when the cells are run.

2. Importing the Datset (10 points)

This portion involves creating your Jupyter notebook and loading the provided dataset into it.

2.1. Steps

  1. Download the provided malware detection dataset file_pe_headers.csv from Blackboard and save it in your project folder.
  2. Run Jupyter Notebook from that folder and create a new notebook.
  3. Use the pandas read_csv function to load the dataset into a DataFrame variable.
  4. Use the shape attribute to print out and verify the dataset’s dimensions.
  5. Use the columns attribute to view the column (feature) names in this dataset.
  6. At the very top of your notebook, before any code, insert a Markdown cell and give it a title “DATA 62500 Lab Project 1”. Also put your name and section number in it.

3. Preparing the data (10 points)

In this part you will create the training and test sets needed to train a classifier.

3.1. Steps

  1. Extract the class column “Malware” from the DataFrame to obtain the label array. Name it y.
  2. Drop the “Malware” and “Name” columns from the DataFrame, and convert the result to a NumPy array. Save the result in the variable X.
  3. Use the train_test_split function to create an 80%/20% train/test split. Use the same variable names shown in the lecture.
  4. Use the shape method to print out the dimensions of the training and test sets, as a basic check that they have been correctly created.

4. Training and evaluating the XGBoost classifier (10 points)

In this portion you will finally see how well an XGBoost classifier can learn to identify malware in this dataset.

4.1. Steps

  1. Import the XGBoost classifier and create a new classifier object.
  2. Use the fit method to train your classifier on the training set only.
  3. Use the predict method to generate predictions for the test set.
  4. Compare your predictions to the true label values for the test set using the accuracy_score function, and print the results.

5. Additional Classifiers (20 points)

This is the most independent part of the project, in which you strike out on your own to get experience training and running classifiers.

5.1. Steps

  1. Consult the scikit-learn documentation (https://scikit-learn.org/stable/user_guide.html) and choose any two other classifiers that can can be used on this (supervised, binary classification) problem.
  2. Train and test a model for each of those two classifiers and print the accuracy results for each. Make sure you set any parameters that may be needed for the model to appropriate values.
  3. Include a markdown cell giving a general description in your own words of the classifiers you chose, and any speculations you may have as to why each performed better or worse than the others. Did any of them beat the XGBoost result?
